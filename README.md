# Customer Churn Analysis using Python

## About the Project

Customer churn is one of the biggest challenges faced by banks and other businesses. Understanding why customers leave can help companies improve their services and retain valuable customers.

In this project, I performed **Exploratory Data Analysis (EDA)** on a bank customer churn dataset using Python. The goal was to clean the data, explore customer behavior, visualize important trends, and identify factors that may be related to customer churn.

This project focuses on understanding the data rather than building a prediction model, making it a solid foundation for future machine learning work.

## Dataset

The dataset contains information about **10,000 bank customers** along with their personal and banking details.

Some of the features include:

- Credit Score
- Geography
- Gender
- Age
- Tenure
- Balance
- Number of Products
- Credit Card Status
- Active Membership
- Estimated Salary
- Satisfaction Score
- Complaint Status
- Card Type
- Reward Points
- Exited (Customer Churn)

## Project Objectives

- Analyze customer demographics and banking behavior.
- Perform data cleaning and quality assessment.
- Explore relationships between customer features and churn.
- Visualize important trends using Python libraries.
- Generate meaningful business insights from the data.

## Key Insights

Some important observations from the analysis are:

- The dataset contains **10,000 customer records** with **no missing values**.
- Around **20% of customers have exited**, while approximately **80% remain with the bank**.
- Features such as **Age, Balance, Active Membership, Satisfaction Score, and Complaint Status** show noticeable differences across customer groups.
- The exploratory analysis provides useful insights that can support customer retention strategies and future predictive modeling.
